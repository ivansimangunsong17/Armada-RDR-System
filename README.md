# Running Discharge Report System

Frontend React.js untuk simulasi input data bongkar muat kapal.

## Teknologi

- React.js
- Vite
- JavaScript
- CSS biasa
- react-router-dom
- Dummy data dan React state

## Menjalankan Project

```bash
npm install
npm run dev
```

Login dummy bisa langsung menggunakan data yang sudah terisi di halaman login.
